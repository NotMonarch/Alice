# Alice â€” Game Bot

> "Alice runs a social deduction game where all players receive the same real question, except for one randomly chosen Imposter who receives a different question. Players answer privately, then vote to identify the Imposter based on the answers revealed anonymously."

Alice is a production-ready Telegram bot built with Python, designed for group chats. It features seamless DM integration, anonymous voting, and persistent leaderboards.


---

## Commands

| Command | Description | Scope | Permission |
| :--- | :--- | :--- | :--- |
| `/start` | Shows the main menu, game explanation, and stats. | Private | Everyone |
| `/newgame [rounds]` | Starts a new game session in a group. Default rounds: 3. | Group | Everyone |
| `/extend` | Adds 30 seconds to the joining timer. | Group | Everyone |
| `/leaderboard` | Displays the top global players by wins. | Group/Private | Everyone |
| `/endgame` | Immediately terminates the current game session. | Group | Admin Only |
| `/addque` | Interactive wizard to add new Questions to the DB. | Private | Sudo Users Only |
